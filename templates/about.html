About Fitture

Fitture is a fitness guide, web-based application which has an image processing function using canny edge detection that will detect what body type the user has based on either the uploaded picture or real-time captured photo of the user’s whole-body portrait. The system will then recommend certain workout exercises and dietary plans based on the predicted body type to help individuals in meeting their fitness goals more accurately. Fitture is always ready to guide you to the fitness route you desire to pursue, no matter where you are.

Body Type

It has a main feature of processing a real-time image that the user can took to predict their body type. The algorithm to be used for the image segmentation is Canny Edge Detection Algorithm (CEDA). The Canny edge detector is an edge detection operator that detects a wide range of edges in images using a multi-stage algorithm.
